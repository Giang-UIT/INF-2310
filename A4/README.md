### How to run the application locally
1) Setup a virtual environments
2) run pip install -r requirements.txt 
3) run *flask run*. 

Make sure to type <ins>localhost."portnumber"</ins> in the search bar instead of 127.0.0.1; otherwise the redirected URI won't work
